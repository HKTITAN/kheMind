# kheMind note templates

Copy a file into `wiki/…`, rename it, replace placeholders (`YYYY-MM-DD`, `Title`, etc.). Frontmatter is optional but helps agents and Dataview-style tools later.

| Template | Use for | Typical path |
|----------|---------|----------------|
| [concept.md](./concept.md) | Ideas, models, definitions | `wiki/concepts/` |
| [person.md](./person.md) | People you track | `wiki/people/` |
| [organization.md](./organization.md) | Companies, teams, communities | `wiki/people/organizations/` |
| [project.md](./project.md) | Initiatives with goals & status | `wiki/projects/` |
| [playbook.md](./playbook.md) | SOPs, prompt chains, checklists | `wiki/playbooks/` |
| [area.md](./area.md) | Life/work areas (PARA “area”) | `wiki/areas/` |
| [reference-book.md](./reference-book.md) | Books | `wiki/references/books/` |
| [reference-paper.md](./reference-paper.md) | Papers | `wiki/references/papers/` |
| [reference-article.md](./reference-article.md) | Web articles, newsletters | `wiki/references/articles/` |
| [reference-course.md](./reference-course.md) | Courses, cohorts | `wiki/references/courses/` |
| [journal-entry.md](./journal-entry.md) | Dated log / reflection | `wiki/journal/YYYY/` |
| [meeting.md](./meeting.md) | Meeting notes | `wiki/meetings/` |
| [decision-adr.md](./decision-adr.md) | Architecture / life decisions | `wiki/decisions/` |
| [resource-tool.md](./resource-tool.md) | Tools, stack items | `wiki/resources/tools/` |
| [resource-link-collection.md](./resource-link-collection.md) | Curated link lists | `wiki/resources/` |
| [question-research.md](./question-research.md) | Open questions, hypotheses | `wiki/questions/` |
| [review-weekly.md](./review-weekly.md) | Weekly review | `wiki/reviews/` |
| [review-quarterly.md](./review-quarterly.md) | Quarterly review | `wiki/reviews/` |
| [moc.md](./moc.md) | Map of content (hub page) | `wiki/moc/` |
| [meta-source-note.md](./meta-source-note.md) | How you use this vault | `wiki/meta/` |

See [wiki/TEMPLATES.md](../wiki/TEMPLATES.md) for conventions and naming.
