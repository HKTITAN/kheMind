# Obsidian

> Example tool page—common pairing with markdown wikis ([Karpathy gist](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)).

## What it does

Local-first markdown editor with graph view, plugins (Dataview, Marp, etc.).

## When I use it

Browsing and light editing of `wiki/` alongside agents.

## Pros

- Files stay plain markdown on disk
- Graph view shows orphan notes

## Cons / limits

- Sync and mobile require deliberate setup
- Not a collaborative CMS out of the box

## Setup notes

- Point vault root at this repo or a subfolder
- Optional: community plugins for wikilinks

## Alternatives

- VS Code, Zed, Neovim for editor-first workflows
