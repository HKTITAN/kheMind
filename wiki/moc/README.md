# moc/

**Maps of content** — hub pages that link out to everything important on a theme. Unlike `_index.md` (global), each MOC is **topic-scoped**.

## Template

[`templates/moc.md`](../../templates/moc.md)

## Naming

`moc-topic-name.md` (prefix `moc-` is optional but grep-friendly).
