# areas/

**Areas** are ongoing responsibilities—they don’t “ship” like projects. Inspired by PARA’s *Areas of responsibility* and Karpathy’s long-horizon themes.

## Template

[`templates/area.md`](../../templates/area.md)

## vs `projects/`

| | Areas | Projects |
|---|--------|----------|
| Horizon | Months–years | Weeks–months |
| Done? | No natural end | Yes (outcomes) |
| Example | Health, career, family | “Launch kheMind docs site” |
